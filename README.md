# Fullstack Project - User Management

## Project Overview
User Management System: signup functional, frontend connected, backend connected.

## Tech Stack
- Backend: Node.js, Express, MySQL, bcrypt
- Frontend: React
- Database: MySQL

## Setup
### Backend
- cd backend
- npm install
- npm start

### Frontend
- cd frontend
- npm install
- npm start

## Database
- `project_db.sql` included
- Table: users
- Trigger: trg_full_name
